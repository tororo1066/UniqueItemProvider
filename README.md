# UniqueItemProvider - ItemInjector設定ガイド（GUI操作向け）

このページは、`ItemInjector` を**インベントリGUIで設定する人向け**の手順書です。
YAMLを直接編集しなくても、基本設定はGUIだけで完結できます。

## 1. まず何ができる？

`ItemInjector` は、次の2つを1セットで作る機能です。

- 条件: どのアイテムに反応するか
- 変更: 反応したアイテムをどう変えるか

プレイヤーがホットバーでアイテムを持ち替えた時に判定され、最初に一致した設定が1つだけ適用されます。

### 注意
- 条件を満たしたアイテムは全て変更されます
- 条件を緩くしすぎたり、変更の設定を入れ忘れたりすると、意図しないアイテムが変わる可能性があります
- 変更のログは残りますが、細心の注意を払って設定してください

## 2. 事前準備

- 権限: `uniqueitemprovider.op`
- コマンドエイリアス: `uip`

## 3. 新規インジェクター作成（GUI中心）

1. `/uip createInjector` を実行
2. 画面左の `条件の設定` をクリック
3. `条件を追加する` から条件タイプを選択
4. 追加した条件アイコンをクリックして値を入力
5. 画面右の `変更の設定` をクリック
6. 変更タイプを選択して、必要項目を入力
7. 下段の `保存する` をクリック
8. 入力ダイアログで以下を入力
   - キー（例: `legendary_sword`）
   - 保存先（例: `weapons/main`）
9. `/uip reload` を実行して反映

保存先を `weapons/main` にした場合、保存ファイルは `plugins/UniqueItemProvider/injectors/weapons/main.yml` です。

## 4. 既存インジェクターの編集

1. `/uip editInjector <key>` を実行
2. `条件の設定` / `変更の設定` を編集
3. `保存する` を押して確定
4. 必要なら `/uip reload` を実行

## 5. 条件設定の操作メモ

### DisplayNameInject

- 用途: アイテム名で判定
- 入力項目:
  - `pattern`（正規表現）
  - `useLegacy`（ON/OFF切替）
- 操作のコツ:
  - 部分一致ではなく**全文一致**です
  - 先頭/末尾を気にせず判定したい場合は `.*` を使ってください

例: `[Legendary]` を含む名前に反応させたい場合

- `.*\[Legendary\].*`

### NBTInject

- 用途: NBT条件で判定
- 入力項目:
  - `requiredNBT`（必要なNBT）
  - `forbiddenNBT`（存在してはいけないNBT）
- 操作のコツ:
  - 入力時にNBTとして解釈できない文字列は保存されません

## 6. 変更設定の操作メモ

### ItemReplaceModification

- 用途: 一致時に別アイテムへ置換
- 操作方法:
  - `変更後のアイテム` をクリック
  - 1スロットGUIに置き換え先アイテムを入れて確定
- 補足:
  - 元の個数をなるべく維持します（置換先の最大スタック数まで）

### AddUpdateProviderModification

- 用途: 一致時に `UpdateProvider` を追加
- 入力項目:
  - `key`

## 7. まず試す最短チェック

1. `DisplayNameInject` だけを設定（例: `.*test.*`）
2. `ItemReplaceModification` でわかりやすいアイテム（例: `DIAMOND_BLOCK`）に設定
3. 保存後に `/uip reload`
4. 名前に `test` を含むアイテムを持ち替えて置換されるか確認

## 8. 反応しない時の確認

1. `保存する` まで完了しているか
2. `/uip reload` を実行したか
3. 条件が厳しすぎないか（特に `DisplayNameInject` の全文一致）
4. `変更の設定` が未設定になっていないか
5. アイテムを「持ち替え」して判定を発生させたか

## 9. 管理者向け補足（最小限）

- インジェクターは `plugins/UniqueItemProvider/injectors/*.yml` から読み込まれます
- 1つのインジェクター内では、複数条件はすべて満たした時のみ適用されます
